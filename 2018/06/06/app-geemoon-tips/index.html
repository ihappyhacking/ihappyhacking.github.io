积木使用小技巧 / GeeMoon Use Tips

发表于 2018-06-06 更新于 2023-04-15 分类于 个人项目 阅读次数：

  • 概览 / Overview

  • 交易 / Transaction
  • 计算器 / Calculator

0x01 概览 / Overview

  • 双击总余额（金额）可以切换金额的隐藏或显示，也可以在“更多”->“显示样式”中开启或关闭。
  • 点击“收入”或“支出”，显示当前周期内的收入或支出的交易列表。
  • 点击账户（列表）进入所有账户列表界面。
  • 趋势默认收支分开显示，可以去“显示样式”中修改，只显示某一个或者合并成一个显示。

0x02 账户 / Account

  • 想要连续浏览某一个账户的所有交易：点击“账户列表” -> 点击“账户”进入详情 -> 点击详情右上角第一个列表按钮来查看，对账时比较方便。也可以使用“高级搜索”来筛选。

0x03 交易 / Transaction

  • 交易列表任意一处长按，可进入多选模式，可以多选删除，或者统计选中的交易的收支金额。

  • 顶部金额输入框右侧上方按钮为选择模板。
  • 顶部金额输入框右侧下方按钮为计算器，可以做加减乘除运算。
  • 分类可以左右滑动选择。
  • 再次点击已选中的分类可取消选中。

0x04 计算器 / Calculator

  • 点击“等号”计算结果或者返回到上一个界面。

0x05 分类 / Category

  • 点击右上角画板按钮，可以选择一键给所有分类设置颜色：
    • 随机：随机选择 App 内置颜色中的一个。
    • 选取颜色：打开颜色选择界面，自己选择一个喜欢的颜色，将所有分类设置为这个颜色。
    • 漂亮模式：只是个人建议，多色图标的分类，颜色会设置为默认的 #EEEEEE，单色或者没有设置图标的分类，会随机一个颜色。ps：跟选取一个颜色类似，多色图标的分类按这个选择设置之后，在报表界面，饼状图就会都显示为一种颜色了，对这个比较在意的，请使用“随机”或者自己修改各个分类的颜色。

0x06 其他 / Others

  • 所有交易列表均支持 3D Touch 预览。
  • 手机桌面，App Icon 也支持 3D Touch 菜单，快速新建交易
