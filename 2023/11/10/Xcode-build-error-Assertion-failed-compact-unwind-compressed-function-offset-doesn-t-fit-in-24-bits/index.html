Xcode 15 项目编译报错：Assertion failed (false && compact unwind compressed function offset doesn't fit in 24 bits), function operator(), file Layout.cpp, line 5758.

发表于 2023-11-10 分类于 iOS 阅读次数：

今天打开了一个挺久前的一个项目，需要增加一点东西，结果发现 Xcode 15 编译运行报错，错误信息看着完全没啥头绪，就去网上搜索，找到的可用的解决办法如下：

在 Targets 的 Build Settings 中搜索 Other Linker Flags, 双击之后添加 -ld64，注意，是小写的 LD，不是 id，然后可以 Clean 一下再重新编译运行。
